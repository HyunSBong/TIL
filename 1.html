HTML

Hypertext Markup Language (HTML)

  1. ol -> orderd list
  2. ul -> unorderd list
  3. li -> list
  4. 타이틀은 title로 무조건 지정해야함 -> 검색엔진을 위함
  5. br -> 줄바꿈
  6. 그러나 단락별로 끊어 줄바꿈을 위한 태그가 존재함 -> p, /p
  7. 이미지를 추가하기 위해서는 img src="~~.jpg", 크기 조절을 위해 width = "~~%"
  8. 한글이 깨지는 것을 방지하기 위해 "meta charset="utf-8"사용
  9. 본문은 body 태그, 본문을 설명하는 Head 태그 그리고 이들을 감싸는 html태그
  10. 또 이 웹페이지가 HTML로서 만들어졌다는 것을 표현하기 위해서 첫줄에 !doctype html 태그를 추가함
  11. target="_blank"는 링크를 클릭했을 때 새창에서 열리도록 하는 속성
