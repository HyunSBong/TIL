CSS

  • CSS 사용을 위해 head 태그안의 style 태그를 넣고 그 안에서 디자인한다. a라고 쓰면 모든 a 태그에 적용한다.
  • 예로 a태그 안의 내용의 색을 바꾸기 위해 font태그를 사용하지 않고 color 메소드를 사용한다.
  • html 문법 안에서 CSS의 문법임을 알리기 위해 style 속성을 사용한다.
  • style 태그 안에서 이름 앞에 .을 붙이면 그 이름의 모든 클래스에 적용하겠다는 것을 의미한다.
  • style 태그 안에서 이름 앞에 #을 붙이면 그 이름의 모든 id에 적용하겠다는 것을 의미한다.
  • 우선순위: 태그 선택자 < 클래스 선택자 < id 선택자
